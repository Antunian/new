# Инструция для работы с Markdown

## Выделение текста 

Чтобы выделить текст курсивом необходимо обрамить его звездочками (*). Например, *вот так*

ЧТобы веделить полужирным, необходимо обрамить его двойныи звездочками(**). Например, **вот так**
